BloodHound AD — Attack Path Cartography

Region: Active Directory • Terrain: Privilege Graph

Instrument: BloodHound
Collector: SharpHound
Engine: Neo4j

BloodHound models Active Directory relationships as navigable terrain. It reveals escalation routes, privilege drift, and lateral movement corridors.

Mission Overview

In AD environments, privilege is rarely linear. BloodHound converts directory objects into graph nodes and relationships into edges, enabling shortest-path privilege analysis.

System Architecture

[ Domain Environment ]

Deployment (Lab)

sudo apt install bloodhound neo4j
sudo neo4j console
bloodhound

Rotate default Neo4j credentials immediately.

Collection (SharpHound)

.\SharpHound.exe -c All

Scoped collection (reduced noise):

.\SharpHound.exe -c Session,LocalAdmin -Throttle 1000 -Jitter 30
