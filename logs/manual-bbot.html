Field Manual — BBOT

Automated reconnaissance orchestration notes (authorized scope, evidence-first outputs).

Scope controls

  • Run only against explicitly authorized targets.
  • Constrain concurrency and rate limits to avoid disruption.
  • Tag and store outputs with source + timestamp for reproducibility.

Output hygiene

  • Normalize findings (domains, subdomains, URLs, IPs) into a single artifact bundle.
  • Deduplicate and annotate confidence/source.
  • Redact sensitive tokens/keys before publishing.

Reinforcement outcomes

  • Asset inventory correction (unknown/forgotten endpoints).
  • Misconfiguration discovery (headers, TLS posture, exposed admin paths).
  • Monitoring recommendations (new host detection, cert transparency alerts).

Public manual omits operational command lines. Keep runbooks privately with client scope and rate limits.
