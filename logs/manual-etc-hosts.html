Field Manual — aircrack-ng

Wireless assessment workflow notes for authorized environments. This manual focuses on scope control, evidence capture, and defensive reporting outcomes.

Scope & Authorization

  • Written authorization and test window defined
  • Target SSIDs/BSSIDs explicitly in scope
  • Rules of engagement include interference limits and safety constraints

Hardware Notes

  • Adapter: Alfa AWUS036AXML (document chipset/driver + OS)
  • Capture environment: location, channel congestion, distance, interference sources

Evidence Capture

  • Record timestamps, channels, BSSID/SSID inventory, and observed security configuration
  • Preserve artifacts (pcap/cap files) with chain-of-custody notes
  • Redact sensitive identifiers when publishing public writeups

Assessment Outcomes

  • Configuration posture: encryption/auth mode, management frame protection status, segmentation
  • Exposure: rogue/evil-twin risk, weak PSK policy indicators, unsafe legacy modes
  • Reinforcement: hardening checklist and monitoring recommendations

Public Disclosure Note

Detailed, step-by-step offensive commands are intentionally omitted from public view. The internal/private version of this manual can include exact command sequences for authorized testing.
